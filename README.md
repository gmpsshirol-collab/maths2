# ಗಣಿತ ಟ್ರ್ಯಾಕರ್ — GitHub ಮೂಲಕ ಆಪ್ ಆಗಿ ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡುವುದು ಹೇಗೆ

## 1) GitHub ಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡಿ
1. github.com ನಲ್ಲಿ ಲಾಗಿನ್ ಆಗಿ (ಖಾತೆ ಇಲ್ಲದಿದ್ದರೆ ಉಚಿತವಾಗಿ ರಚಿಸಿ)
2. ಮೇಲಿನ **+** ಬಟನ್ → **New repository** ಒತ್ತಿ
3. ಹೆಸರು ಕೊಡಿ, ಉದಾ: `ganita-tracker` → **Public** ಆಯ್ಕೆಮಾಡಿ → **Create repository**
4. ಈಗ ತೆರೆದ ಪೇಜ್‌ನಲ್ಲಿ **"uploading an existing file"** ಲಿಂಕ್ ಒತ್ತಿ
5. ಈ 4 ಫೈಲ್‌ಗಳನ್ನು ಡ್ರ್ಯಾಗ್ ಮಾಡಿ ಬಿಡಿ: `index.html`, `manifest.json`, `service-worker.js`, `icon-192.png`, `icon-512.png`
6. ಕೆಳಗೆ **Commit changes** ಒತ್ತಿ

## 2) GitHub Pages ಆನ್ ಮಾಡಿ
1. ರೆಪೊಸಿಟರಿಯ **Settings** ಟ್ಯಾಬ್‌ಗೆ ಹೋಗಿ
2. ಎಡಬದಿಯ **Pages** ಕ್ಲಿಕ್ ಮಾಡಿ
3. **Branch** ಅಡಿಯಲ್ಲಿ `main` ಆಯ್ಕೆಮಾಡಿ, ಫೋಲ್ಡರ್ `/ (root)` ಎಂದೇ ಇಟ್ಟು **Save** ಒತ್ತಿ
4. 1-2 ನಿಮಿಷದಲ್ಲಿ ಮೇಲೆ ಒಂದು ಲಿಂಕ್ ಕಾಣಿಸುತ್ತದೆ:
   `https://<ನಿಮ್ಮ-ಯೂಸರ್ ಹೆಸರು>.github.io/ganita-tracker/`

## 3) ಮೊಬೈಲ್‌ನಲ್ಲಿ ಆಪ್ ಆಗಿ ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ
ಆ ಲಿಂಕ್ ಅನ್ನು ಫೋನ್‌ನ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ತೆರೆಯಿರಿ:

**ಆಂಡ್ರಾಯ್ಡ್ (Chrome):**
- ಪೇಜ್ ತೆರೆದ ಸ್ವಲ್ಪ ಸಮಯದಲ್ಲೇ ಕೆಳಗೆ/ಮೇಲೆ **"Install app"** ಎಂಬ ಬ್ಯಾನರ್ ತಾನಾಗಿಯೇ ಬರುತ್ತದೆ — ಒತ್ತಿದರೆ ಆಪ್ ಇನ್‌ಸ್ಟಾಲ್ ಆಗುತ್ತದೆ
- ಬಾರದಿದ್ದರೆ, ⋮ ಮೆನು → **"Install app"** ಅಥವಾ **"Add to Home screen"**

**iPhone (Safari):**
- Share ಬಟನ್ → **"Add to Home Screen"**

ಇನ್‌ಸ್ಟಾಲ್ ಆದ ಮೇಲೆ ಹೋಮ್ ಸ್ಕ್ರೀನ್‌ನಲ್ಲಿ ✏️ ಐಕಾನ್ ಕಾಣಿಸುತ್ತದೆ, ಬ್ರೌಸರ್ ಬಾರ್ ಇಲ್ಲದೆ ಪೂರ್ಣ ಆಪ್‌ನಂತೆ ತೆರೆಯುತ್ತದೆ, ಇಂಟರ್ನೆಟ್ ಇಲ್ಲದಿದ್ದರೂ (ಆಫ್‌ಲೈನ್) ಮತ್ತೆ ತೆರೆಯಬಹುದು.

## ಗಮನಿಸಿ
- ಡೇಟಾ (ವಿದ್ಯಾರ್ಥಿಗಳ ಪ್ರಗತಿ) ಆ ಫೋನ್/ಬ್ರೌಸರ್‌ನ ಸ್ಥಳೀಯ ಸಂಗ್ರಹಣೆ (localStorage) ನಲ್ಲಿ ಇರುತ್ತದೆ — ಬೇರೆ ಫೋನ್‌ನಲ್ಲಿ ತೆರೆದರೆ ಅಥವಾ ಬ್ರೌಸರ್ ಡೇಟಾ ಕ್ಲಿಯರ್ ಮಾಡಿದರೆ ಡೇಟಾ ಕಾಣಿಸುವುದಿಲ್ಲ.
- ರೆಪೊಸಿಟರಿ **Public** ಆಗಿಟ್ಟರೆ ಮಾತ್ರ GitHub Pages ಉಚಿತವಾಗಿ ಸಿಗುತ್ತದೆ (ಯಾರಾದರೂ ಲಿಂಕ್ ನೋಡಬಹುದು, ಆದರೆ ವಿದ್ಯಾರ್ಥಿಗಳ ಡೇಟಾ ಪ್ರತಿಯೊಬ್ಬ ಬಳಕೆದಾರರ ಸ್ವಂತ ಫೋನ್‌ನಲ್ಲೇ ಇರುತ್ತದೆ, ಸರ್ವರ್‌ನಲ್ಲಿ ಅಲ್ಲ).
